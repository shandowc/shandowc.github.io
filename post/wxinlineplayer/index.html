WXInlinePlayer是一个使用wasm软解视频的播放器，支持h264/h265播放。今天我们阅读一下他们的源码，主要以增加代码注释以及画图等方式加深理解。

JavaScript部分

解码和解封装都是在wasm中完成的，为了方便，后面都将这个解码解封装器叫做解码器，也就是代码中的H264Codec，虽然叫H264，但是不止能解码H264，还能解封装FLV、解码H265。

Boolean WXInlinePlayer.isSupport(void)

  • 当前执行环境是否支持WXInlinePlayer

Promise WXInlinePlayer.init(Object)

  • 传入参数有两个字段asmUrl, wasmUrl，通过判断是否只是wasm决定加载哪个脚本
  • 通过操作dom，append一个scripte标签，导入解码器wasm
  • 在script的onload事件中，将WXInlinePlayer.isInited设置为true

Promise WXInlinePlayer.ready(void)

  • 检查是否init完成

void WXInlinePlayer._initlize(void)

  • 设置一个250毫秒更新一次的this.timeUpdateTimer，用于上报更新时间事件
  • 初始化一个this.drawer，用来绘制视频画面
  • 初始化一个this.loader，用来下载视频数据，支持传入自定义的loader
  • 将this.loader的loadError和loadSuccess透传到外部事件侦听器
  • 初始化一个this.processor并监听事件

void WXInlinePlayer._onMediaInfoHandler(Ojbect)

  • 从回调中获得this.duration, this.width, this.height
  • 自身再emit这个事件，让外部可以获得这个数据

void WXInlinePlayer._onFrameHandler(Object)

  • 获得视频帧宽、高、YUV数据
  • 调用this.drawer.drawNextOutputPicture(width, height, data)画图

void WXInlinePlayer._onBufferingHandler(void)

  • 设置状态为buffering
  • 抛出buffer事件，前端可能显示一个缓冲提示
  • 调用this.loader.read读取数据喂给this.processor.process
  • 可以看到，这个事件this.processor拿到数据，并进而传递给解码器的一种方式

void WXInlinePlayer._onPreloadHandler(void)

  • 和buffering事件一样，调用this.loader.read读取数据喂给this.processor.process
  • 这个事件不会改变播放器状态，是预加载视频的一个方式，只改变了processor的状态，等于是一个内部状态
  • 通过这种方式让播放器受网络波动影响小一些

void WXInlinePlayer._onPlayingHandler(void)

  • 当播放器不处于播放状态时，设置为playing状态
  • 抛出playing状态
  • 是一种从buffering转变成playing的方式

void WXInlinePlayer._onEndHandler(void)

  • 设置播放器isEnd为true
  • 抛出end事件

void WXInlinePlayer.play(void)

  • 调用this._initlize初始化播放
  • 调用this.processor.unblock
  • 抛出play事件

void WXInlinePlayer.pause()

  • 直播流不支持暂停
  • 调用processor的pause方法
  • 抛出paused事件

void WXInlinePlayer.resume(void)

  • 直播流没有pause所以不用resume
  • 其实是透传到this.processor.resume

Number|void WXInlinePlayerInstance.volume(Number|void)

  • 获取/设置当前音量
  • 其实是透传到this.processor.volume

Boolean|void WXInlinePlayerInstance.mute(Boolean|void)

  • 获取/设置静音状态
  • 其实是透传到this.processor.mute

void WXInlinePlayerInstance.destroy(void)

  • 销毁播放器，释放所有内存等待回收
  • 调用this.removeAllListeners，移除事件侦听
  • 调用this.stop停止播放器
  • 调用this.drawer.destroy销毁绘制器

Number WXInlinePlayerInstance.getCurrentTime(void)

  • 获取当前播放时间
  • 其实是透传this.processor.getCurrentTime()

Number WXInlinePlayerInstance.getAvaiableDuration(void)

  • 获得可播放时长
  • 其实是透传this.processor.getAvaiableDuration()

Number WXInlinePlayerInstance.getDuration()

  • 获得流媒体总时长
  • 直接返回this.duration
  • this.duration的值来自this.processor上报的mediaInfo

    volume = 1.0,
    muted = false,
    preloadTime = 1000,
    bufferingTime = 3000,
    cacheSegmentCount = 128
  }) {
    super();
    this.averageUnitDuration = 0;     // 平均解码得到的可播放时长
    this.averageDecodeCost = 0;       // 平均解码消耗时间
    this.soundHeadSliced = false;     // 初始音频数据是否已经到达（目的不明确TODO）
    this.framerate = 1000 / 24;       // 默认每秒约41.6帧，后面会根据解码回调，调整帧率
    this.isEnded = false;             // 解码返回complete
    this.state = 'created';           // 播放状态
    this.baseTime = 0;                // 开始播放的起始时间戳
    this.blocked = !Util.isWeChat();  
    this.hasVideo = true;             // 是否有视频流
    this.hasAudio = true;             // 是否有音频流
    this.frames = [];                 // 视频帧队列
    this.audios = [];                 // 音频帧队列
    this.currentTime = 0;             // 时间戳，保证初始时间肯定是0，然后单调递增的毫秒时间戳
    this.bufferingIndex = -1;         // 开始缓存的frame下标，播放的时候需要frames的最新的时间戳减去这个下标frame时间戳大于最小缓冲时间
    this.minBufferingTime = preloadTime;        // 最小缓冲时间，开始时值为proloadTtime，后面为bufferingTime
    this.bufferingTime = bufferingTime;         // 缓冲时间
    this.cacheSegmentCount = cacheSegmentCount; // 调整ticker时间的依据，当frames的值小于或者大于这个值的一定比率后，相应调整这个值

    this.sound = new Sound({ volume, muted });  // 封装的sound部分代码

    this.codec.onmessage = this._onCodecMsgHandler.bind(this);

然后从函数整体概览一下processor。

Number Processor.getAvaiableDuration(void)

  • 有音频则从this.sound获取
  • 否则返回this.frames最后一帧的时间戳

Number Processor.getCurrentTime(void)

  • 毫秒为单位的现在时间
  • 有音频从this.sound.getCurrentTime()获取
  • 否则返回this.currentTime

void Processor.process(Uint8Array buffer)

  • 调用this.codec解码

void Processor.unblock(void)

  • 调用this.sound.unblock(0);

Number|void Processor.volume(Number|void)

  • 透传到this.sound.volume方法

Boolean|void Processor.mute(Boolean|void)

  • 透传到this.sound.mute方法

void Processor.pause(void)

  • 调用this.sound.pause暂停音频
  • 调用this.ticker.remove停止定时回调this._onTickHandler函数
  • 设置状态为paused

void Processor.resume(void)

  • 调用this.sound.resume恢复音频
  • 调用this.ticker.add恢复定时调用this.tickHandler
  • 设置状态为playing

void Processor.destroy(void)

  • 调用this.removeAllListeners移除所有事件监听器
  • 调用this.sound.destroy移除音频组件
  • 调用this.codec.destroy销毁codec
  • 初始化成员变量，状态设置为desctroy

void Processor._onTickHandler(void)

  • 通过this.emit('buffering')表明数据不足以维持播放，请求外部喂数据
  • 通过this.emit('preload')请求加载数据，避免网络波动导致播放卡顿
  • 通过this.ticker.setFps设置本函数回调速度
  • 通过this.emit('frame', this.frames[i]);让外部渲染在currentTime 25毫秒内的帧画面

void Processor._onCodecMsgHandler(MessageEvent)

  • 解码器回调接收函数
  • 收到ready消息，this.emit('buffering')
  • 收到header消息，判断是否有音视频，然后自身emit透传这个事件
  • 收到mediaInfo消息
    • 设置this.framerate,
    • 设置this.ticker定时间隔
    • 自身emit透传这个事件
  • 收到video消息
    • 拿到视频帧YUV数据，宽，高，pts时间戳
    • 如果this.baseTime未设置，设置为拿到的视频帧的时间戳
    • 将时间戳设置为timestamp=timestamp-this.baseTime，放入this.frames
  • 收到audio消息
    • 拿到音频时间戳和音频aac数据
    • 如果this.baseTime未设置，设置为拿到的时间戳
    • 将数据放入this.audios
  • 收到decode消息
    • 拿到每次解码耗时、以及解码可播放时长
    • this.emit('playing')，并设置状态为playing
    • emit performance事件，上报平均解码耗时和平均解码可播放时长
  • 收到complete消息
    • 设置状态为end
    • emit end事件

  _request() {
    let isSuccess = false;
    let promise = this._fetch();
    for (let i = 0; i < MAX_REQ_RETRY; i++) {
      promise = promise
        .then(buffer => {
          if (!isSuccess) {
            isSuccess = true;
            this.downloadSize += buffer.length;
            if (buffer.length < this.chunkSize) {
              this.done = true;         // 获取到的数据不足chunk大小，表明文件下载完了
            }
          }

          if (supportSharedBuffer) {    // 支持sharedBuffer的浏览器就用，避免后续多次拷贝
            let sharedBuffer = new SharedArrayBuffer(buffer.byteLength);
            let result = new Uint8Array(sharedBuffer);
            result.set(buffer);
            buffer = result;
          }

          return buffer;
        })
        .catch(e => {
          if (i >= MAX_REQ_RETRY - 1) {
            if (!this.emitted) {              // 只会抛出事件一次
              this.emitted = true;
              this.emit('loadError', e);      // 失败次数超过限制，抛出事件
            }
            throw e;
          } else {
            return this._fetch();
          }
        });
    }
    return promise;
  }

this._request会调用this._fetch去获取数据，并在失败的时候重试MAX_REQ_RETRY次。

  _fetch() {
    return new Promise((resolve, reject) => {
      const endIndex = this.startIndex + this.chunkSize;
      this.xhr = new XMLHttpRequest();
      this.xhr.open('GET', this.url);
      this.xhr.responseType = 'arraybuffer';
      this.xhr.setRequestHeader(
        'Range',
        `bytes=${this.startIndex}-${endIndex}`    //通过range控制每次只下载chunk大小的数据
      );

      this.xhr.onerror = e => {
        reject({
          status: -1,
          statusText: 'unknown error',
          detail: e
        });
      };

      this.xhr.onload = () => {
        if (this.xhr.readyState == 4) {
          if (this.xhr.status >= 200 && this.xhr.status <= 299) {
            if (!this.emitted) {            // 只会抛出事件一次
              this.emitted = true;
              this.emit('loadSuccess');
            }
            this.startIndex = endIndex + 1;
            resolve(new Uint8Array(this.xhr.response));
          } else {
            reject({
              status: this.xhr.status,
              statusText: this.xhr.statusText,
              detail: String.fromCharCode.apply(
                null,
                new Uint8Array(this.xhr.response)
              )
            });
          }
        }
      };
      this.xhr.send();
    });
  }

可以看到，chunk.js的方法是通过range方法每次下载chunk大小的数据，失败的时候会重试一定次数。

  getFuncBody(funcStr) {
    return funcStr
      .trim()
      .match(/^function\s*\w*\s*\([\w\s,]*\)\s*{([\w\W]*?)}$/)[1];
  }

  workerify(func, methods = []) {
    // StreamLoader是将全部代码都放到一个function里面，getFuncBody则从这个函数的
    // 字符串中，提取函数内部的代码，从而拿到一个完整的js文件
    const funcStr = this.getFuncBody(func.toString());
    function __Worker__(data) {
      EventEmitter.call(this);
      this.id = 0;
      this.resolves = [];

      const blob = new Blob([funcStr], { type: 'text/javascript' }); // 用函数体代码创建一个Blob
      this.url = URL.createObjectURL(blob);   // 给这个blob创建对象URL
      this.worker = new Worker(this.url);     // 创建web worker
      this.worker.onmessage = message => {    // worker完成工作后，返回结果
        const { id, data, destroy, type } = message.data;
        if (destroy) {
          this.resolves = [];
          URL.revokeObjectURL(this.url);
          this.worker.terminate();
          this.worker = null;
        } else if (type == 'event') {
          this.emit(data.type, data.data);    // 对于Streamloader，是转发loadError和loadSuccess事件
        } else {
          for (let i = this.resolves.length - 1; i >= 0; i--) {
            if (id == this.resolves[i].id) {
              this.resolves[i].resolve(data);         // 在这里resolve之前调用method返回的promise
              this.resolves.splice(i, 1);
              break;
            }
          }
        }
      };

      this.worker.postMessage({ type: 'constructor', id: this.id++, data });
    }

    inherits(__Worker__, EventEmitter);

    for (let i = 0; i < methods.length; i++) {
      const type = methods[i];
      __Worker__.prototype[type] = function(data) {      // 让本对象实现methods方法
        return new Promise((resolve, reject) => {        // 返回的是Promise，因为是在worker中完成的，无法立刻返回结果
          const id = this.id++;
          this.resolves.push({ id, resolve, reject });
          if (this.worker) {
            this.worker.postMessage({ type, id, data });  // 通过postmessage让worker去执行具体动作
          }
        });
      };
    }

    return __Worker__;
  }

  function slice(buffer, startIndex, endIndex) {
    if (!endIndex || endIndex - startIndex > buffer.length) {
      endIndex = buffer.length;
    }

    if(supportSharedBuffer){          // 支持sharedArray的时候用它减少后续拷贝
      let sharedBuffer = new SharedArrayBuffer(endIndex - startIndex);
      let result = new Uint8Array(sharedBuffer);
      result.set(buffer.subarray(startIndex, endIndex));
      return result;
    }else{
      return buffer.subarray(startIndex, endIndex);
    }
  }

  StreamLoader.prototype._getChunk = function() {
    return new Promise(resolve => {
      const buffer = slice(this.data, 0, this.chunkSize);
      this.data = slice(
        this.data, 
        this.data.length <= this.chunkSize ? this.data.length : this.chunkSize
      );
      resolve(buffer);
    });
  };

  function concat(i, j) {
    const buffer = new Uint8Array(i.length + j.length);
    buffer.set(new Uint8Array(i), 0);
    buffer.set(new Uint8Array(j), i.length);
    return buffer;
  }

  StreamLoader.prototype._request = function() {
    if (this.reader) {                   // 看之前请求过没，请求过就能拿到reader
      return this.reader.read().then(result => {
        let { value, done } = result;
        value = new Uint8Array(value ? value : 0);
        this.data = concat(this.data, value);   // 将数据拼到this.data里面
        if (done) {
          this.done = true;                 // 是否遇到EOF
        } else if (this.data.length < this.chunkSize) {
          return this._request();           // 读到数据，但是数据不足，递归调用自己
        }
      });
    } else {
      return fetch(this.url, {          // 只在第一次调用fetch，拿到reader
        method: 'GET'
      })
        .then(resp => {
          const { status, statusText } = resp;
          if (status < 200 || status > 299) {
            return resp.text().then(text => {
              self.postMessage({
                type: 'event',
                data: {
                  type: 'loadError',            // 状态码不是2xx就报错，是就返回加载成功
                  data: { status, statusText, detail: text }
                }
              });
            });
          }

          self.postMessage({ type: 'event', data: { type: 'loadSuccess' } });
          this.reader = resp.body.getReader();
          return this._request();
        })
        .catch(e => {
          self.postMessage({
            type: 'event',
            data: {
              type: 'loadError',           // fetch失败也报错
              data: { status: -1, statusText: 'unknown error', detail: e }
            }
          });
        });
    }
  };

  decode(data) {
    if (data.length) {
      data = Buffer.from(data);
      this.data = Buffer.concat([this.data, data]); // 将数据放到this.data中保存起来
      if (this.context) {
        return new Promise(resolve => {
          this.context.decodeAudioData(       // 调用AudioContext.decodeAudioData()
            this.data.buffer,
            buffer => {
              this._onDecodeSuccess(buffer);
              resolve();
            },
            error => {
              this._onDecodeError(error);
              resolve();
            }
          );
        });
      }
    }
    return Promise.resolve();
  }

  _onDecodeSuccess(audioBuffer) {
    const audioSrc = this.context.createBufferSource();
    audioSrc.onended = this._onAudioBufferEnded.bind(this);   // 播放完及时清理this.audioSrcNodes

    if (!this.playStartedAt) {
      const { duration } = audioBuffer;       // 第一段音频数据的时长
      const { currentTime, baseLatency, sampleRate } = this.context;
      const startDelay = duration + (baseLatency || 128 / sampleRate); // 第一段数据延迟一定时间再播放
      this.playStartedAt = currentTime + startDelay;                   // 记录开始播放的时间
    }

    audioSrc.buffer = audioBuffer;
    if (this.state == 'running') {
      try {
        audioSrc.connect(this.gainNode);
        audioSrc.start(this.totalTimeScheduled + this.playStartedAt); // 通过时间轴，指定每段音频的播放开始时间，避免卡顿
      } catch (e) {}
    }

    this.audioSrcNodes.push({
      source: audioSrc,
      duration: audioBuffer.duration,         // 这段音频的时长
      timestamp: this.totalTimeScheduled      // 这段音频在所有可播放的音频段中的时间位置
    });

    this.totalTimeScheduled += audioBuffer.duration;
    this.duration += audioBuffer.duration;

    this.data = Buffer.alloc(0);
    this.emit('decode:success');
  }

  _onDecodeError(e) {
    this.emit('decode:error', e);       // 音频解码失败返回事件
  }

  unblock(offset) {
    if (this.state != 'blocked') {
      return;
    }

    this.state = 'running';
    this.resume();
    this.setBlockedCurrTime(offset);

    this.playStartedAt = 0;     // 初始化时间变量状态
    this.totalTimeScheduled = 0;
    for (let i = 0; i < this.audioSrcNodes.length; i++) {
      const { source, timestamp, duration } = this.audioSrcNodes[i];
      source.onended = null;
      source.disconnect();

      const audioSrc = this.context.createBufferSource();
      audioSrc.onended = this._onAudioBufferEnded.bind(this);   // 清理播放完的this.audioSrcNodes
      if (!this.playStartedAt) {
        const { currentTime, baseLatency, sampleRate } = this.context;
        const startDelay = duration + (baseLatency || 128 / sampleRate);
        this.playStartedAt = currentTime + startDelay;
      }

      audioSrc.buffer = source.buffer;
      try {
        audioSrc.connect(this.gainNode);
        audioSrc.start(
          this.totalTimeScheduled + this.playStartedAt,   // 给音频安排schedule，让它在适当的时间播放
          !i ? offset / 1000 - timestamp : 0
        );
      } catch (e) {}

      this.audioSrcNodes[i].source = audioSrc;
      this.audioSrcNodes[i].timestamp = this.totalTimeScheduled;
      this.totalTimeScheduled += duration;
    }
  }

  drawNextOutputPicture(width, height, data) {
    const { yTextureRef, uTextureRef, vTextureRef } = this;
    const gl = this.contextGL;
    gl.pixelStorei(gl.UNPACK_ALIGNMENT, 1);     // 每一行像素数据下标无需对齐

    // 告诉WebGL裁剪空间的-1 -> +1 分别对应到x轴的0 -> width和y轴的0 -> height
    gl.viewport(0, 0, width, height);

    const i420Data = data;                      // 传进来的是YUV420P的数据
    const yDataLength = width * height;         // Y数据的长度是宽x高
    const yData = i420Data.subarray(0, yDataLength);  // 取本帧中Y的数据
    gl.activeTexture(gl.TEXTURE0);              // 激活Y纹理单元
    gl.bindTexture(gl.TEXTURE_2D, yTextureRef); // 绑定一个纹理，GL 最多可同时注册32张纹理
    gl.texImage2D(                              // 加载图片数据到Y纹理中
      gl.TEXTURE_2D,
      0,
      gl.LUMINANCE,
      width,
      height,
      0,
      gl.LUMINANCE,
      gl.UNSIGNED_BYTE,
      yData
    );

    const cbDataLength = width * height / 4;    // uv数据的长度
    const cbData = i420Data.subarray(yDataLength, yDataLength + cbDataLength);
    gl.activeTexture(gl.TEXTURE1);              // 激活U纹理单元
    gl.bindTexture(gl.TEXTURE_2D, uTextureRef); // 绑定U纹理到当前GL
    gl.texImage2D(                              // 加载图片数据到U纹理中
      gl.TEXTURE_2D,
      0,
      gl.LUMINANCE,
      width / 2,
      height / 2,
      0,
      gl.LUMINANCE,
      gl.UNSIGNED_BYTE,
      cbData
    );

    const crDataLength = cbDataLength;          //uv数据长度
    const crData = i420Data.subarray(
      yDataLength + cbDataLength,
      yDataLength + cbDataLength + crDataLength
    );

    gl.activeTexture(gl.TEXTURE2);              // 激活V纹理单元
    gl.bindTexture(gl.TEXTURE_2D, vTextureRef); // 绑定V纹理到当前GL
    gl.texImage2D(                              // 加载图片数据到V纹理中
      gl.TEXTURE_2D,
      0,
      gl.LUMINANCE,
      width / 2,
      height / 2,
      0,
      gl.LUMINANCE,
      gl.UNSIGNED_BYTE,
      crData
    );

    gl.drawArrays(gl.TRIANGLE_STRIP, 0, 4);     // 根据gl_Position的顶点, 通过绘制两个有共享边的三角形的方式绘制一个矩形
  }
